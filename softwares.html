  • Unsupervised Distance Learning Framework (UDLF)

    The UDLF is an open-source software developed and maintained by me under the supervision of Prof. Dr. Daniel C. G. Pedronette. The software currently consists in the implementation of seven different unsupervised learning methods for image retrieval. You can consult my publication and the official page for more information.

    VALEM, L. P. ; PEDRONETTE, D. C. G. . An Unsupervised Distance Learning Framework for Multimedia Retrieval. In: International Conference on Multimedia Retrieval (ICMR) (accepted), 2017.
